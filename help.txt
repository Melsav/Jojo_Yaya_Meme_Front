Pour aligner uniquement les deux derniers boutons tout en gardant le reste des éléments (comme l'input de type date) dans leur position initiale, et placer spécifiquement le bouton "Revenir à l'accueil" à l'extrême gauche de la page, tout en maintenant les autres éléments centrés, vous pouvez utiliser une combinaison de Flexbox et de positionnement absolu ou d'autres techniques de disposition spécifiques.

Une approche serait de créer un conteneur supplémentaire pour les deux boutons que vous souhaitez aligner séparément et de gérer leur positionnement de manière indépendante. Voici une mise à jour de votre code pour refléter cela :
HTML Modifié

html

<div id="controls">
    <div id="leftButton">
        <button id="revenir" onclick="goToHomePage()">Revenir à l'accueil</button>
    </div>
    <input type="date" id="datePicker">
    <div id="rightButtons">
        <button onclick="fetchAPOD()">Show</button>
        <button id="random" onclick="fetchRandomAPOD()">Random Picture</button>
    </div>
</div>

CSS Modifié

css

#controls {
    display: flex;
    flex-direction: column;
    align-items: center;
    width: 100%;
}

#leftButton, #rightButtons {
    width: 100%;
    display: flex;
    justify-content: space-between;
}

#leftButton {
    position: absolute;
    left: 20px; /* Ajustez en fonction de la marge intérieure de votre body ou conteneur parent */
    top: 0;
}

#rightButtons {
    justify-content: center;
    gap: 10px; /* Espacement entre les boutons "Show" et "Random Picture" */
}

button, input[type="date"] {
    padding: 10px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 16px;
    outline: none;
}

button {
    background-color: #08d;
    color: white;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: #06b;
}

input[type="date"] {
    background-color: #fff;
    color: #555;
    margin: 20px 0; /* Assurez-vous que l'input est bien espacé des boutons */
}

@media (max-width: 600px) {
    #controls, #leftButton, #rightButtons {
        flex-direction: column;
        position: static; /* Annule le positionnement absolu pour le responsive */
        align-items: center; /* Centre les éléments dans la vue mobile */
    }

    #leftButton, #rightButtons {
        justify-content: center; /* Centre les boutons dans leurs conteneurs respectifs */
    }

    input[type="date"], button {
        width: 100%;
    }
}

Cette solution positionne le bouton "Revenir à l'accueil" à l'extrême gauche en utilisant position: absolute; et ajuste son positionnement horizontal avec left: 20px;. Les boutons "Show" et "Random Picture" sont regroupés dans leur propre div (#rightButtons) et centrés ensemble. L'input de type date reste centré au-dessus de ces boutons.

Notez que l'utilisation de position: absolute; pour #leftButton nécessite une gestion attentive du positionnement et peut nécessiter des ajustements supplémentaires en fonction de la structure globale de votre page et de ses marges/paddings. Les media queries ajustent le comportement pour les écrans plus petits en annulant le positionnement absolu et en revenant à une disposition plus traditionnelle centrée.
